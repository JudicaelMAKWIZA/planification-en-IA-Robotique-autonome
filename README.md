# Planification des tâches pour un robot mobile autonome

Dépôt compagnon au rapport de séminaire : *Planification des tâches pour un robot mobile autonome*.

Ce projet regroupe la documentation, les modèles PDDL/STRIPS, et les implémentations de référence liées au rapport :
- Modélisation (domain.pddl / problem.pddl)
- Planification symbolique (STRIPS / PDDL)
- Planification de navigation (A* sur grille)
- Scripts d'exécution et visualisations (à venir)

## Organisation prévue
- `docs/` — Rapport, diagrammes, notes
- `pddl/` — Domaines et instances PDDL
- `planner/` — Code du planificateur (prototype)
- `nav/` — Algorithme A* et simulations réseau (prototype)
- `examples/` — Scénarios de test

## Roadmap
1. Déposer les fichiers PDDL et les diagrammes (phase 1)
2. Intégrer les scripts prototypes (phase 2 — implémentation)
3. Ajouter visualisations et exemples exécutables (phase 3)

*Le contenu complet (code et exemples) sera ajouté en complément du rapport.*
